# Collaborator werden

Wir suchen aktive und motivierte Leute, die beim Projekt mithelfen möchten.

Egal ob du:
- programmieren kannst
- Webseiten designen möchtest
- Ideen für Features hast
- Browser-Games entwickeln willst
- Bugs finden möchtest
- bei Organisation helfen willst
- oder einfach kreativ bist

Du kannst dich als Collaborator bewerben.

## Voraussetzungen

Du solltest:
- ein GitHub-Konto besitzen
- freundlich und respektvoll sein
- keine schädlichen Dateien hochladen
- keine Inhalte stehlen
- mit anderen Mitgliedern zusammenarbeiten können
- Spaß an Projekten und Entwicklung haben

## So bewirbst du dich

Schreibe eine E-Mail an:

**auron-team@proton.me**

Die Nachricht sollte enthalten:

1. Deinen GitHub-Namen
2. Dein Alter (optional)
3. Was du gut kannst
4. Warum du beim Projekt helfen möchtest
5. Welche Ideen du für das Projekt hast
6. Ob du bereits Erfahrung mit Coding hast
7. Links zu eigenen Projekten (optional)

## Schnellbewerbung

Nutze die vorbereitete Mail-Vorlage:

[MAIL_TEMPLATE.md](./MAIL_TEMPLATE.md)

## Was Collaborator machen dürfen

Collaborator können je nach Rolle:
- Dateien bearbeiten
- neue Features entwickeln
- Pull Requests erstellen
- Issues bearbeiten
- Bugs fixen
- Ideen diskutieren
- neue Repositories mitentwickeln
- Browser Games erstellen
- KI-Projekte erweitern

Nicht jeder bekommt sofort vollständigen Zugriff. Manche Mitglieder erhalten zuerst eingeschränkte Rechte.

## Regeln

Folgende Dinge sind verboten:
- Spam
- beleidigende Inhalte
- Malware
- zerstören von Dateien
- unerlaubtes Leaken von Inhalten
- absichtliches Sabotieren des Projekts

Wer gegen Regeln verstößt, kann entfernt werden.

## Ziel des Projekts

Das Ziel dieses Projekts ist es, coole Dinge zu bauen, neue Ideen auszuprobieren und gemeinsam kreative Projekte zu entwickeln.

Geplant sind unter anderem:
- KI-Tools
- Browser Games
- Web Apps
- Experimente
- Automatisierungen
- coole Designs
- Community-Projekte

Vielen Dank an alle, die das Projekt unterstützen.
